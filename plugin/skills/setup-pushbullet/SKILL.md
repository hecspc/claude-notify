# Setup Pushbullet Notifications

Run `claude-notify setup pushbullet <API_TOKEN>` to configure Pushbullet as a notification backend.

## How to get an API token

1. Go to [Pushbullet Settings](https://www.pushbullet.com/#settings/account)
2. Create an Access Token

## Usage

The user provides a Pushbullet API token as the argument.

Example: `claude-notify setup pushbullet o.xxxxxxxxxxxxxxxxxxxxx`

Run the command with the user's API token, then show the output.
